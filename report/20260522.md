## go-yaml

- [ ] [PR #353 feat(node): add ability to merge mapping nodes (node.merge())](https://github.com/yaml/go-yaml/pull/353)
  A new open PR implementing a `Merge()` method for `*yaml.Node` mappings, allowing keys from a source node to be merged into a destination rather than the whole node being replaced. This is the first concrete implementation of the long-running feature request in issue #289, opened by the same author on the same day.
