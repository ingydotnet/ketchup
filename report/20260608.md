## yq

- [ ] [PR #2730 fix: preserve correct parent references in explode merge anchor reconstruction](https://github.com/mikefarah/yq/pull/2730)
  Active again today (2026-06-08): the fix for stale parent pointers in yq's `--yaml-fix-merge-anchor-to-spec` merge reconstruction path advanced again, with `updated_at` moving from 2026-06-06 to 2026-06-08. Closes issue #2434.
